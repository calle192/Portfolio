I Positioned With Visibility

I chose to use the visibility attribute. Origionally I wanted to use z.index because it was similiar to the sort of programming I've used in the past with C# and java. zindex uses a numeric value to decide which image is on top. Generally speaking thats how I make most functions/ methods that contain loops or make decisions. I wanted to do something different this time. Visiblity is technically simpilar that zindex. All you have to do is assign a value of either hidden or visible. you do this in the javascript function by declaring getElementById(idName).style.visibility = "value";. You place either "hidden" or "visible" where I wrote value and that controls which image is seen.
